1. Extract the .zip file
2. To run the application, open lms_db_project.py file
3. To execeute from the command line: python lms_db_project.py
4. Connection to the database PostgreSQL has been made using the following credentials:
	{ host="localhost",  user="postgres",  password="12345",  database="lms" }
5. Once connection has been made, database is ready to use.
6. The GUI will be launched.

Dependencies: Any version of Python which supports TKinter library and PostgreSQL. 
	      Recommended to use version 3.6.5 or later.
